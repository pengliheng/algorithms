### 简单算法面经
- [ ] 在无序数据结构中,快速寻找中位数

简单的快排,找中间,复杂度是O(nlogn)
快排的优化空间,取中间,将少的另一部分舍弃,继续在大的区间取中间,记得把前面的舍弃的位数记上,,如果运气好,左右相等,则选的基点就是中位数,,,快排的灵活运用啊....

- [ ] 为什么快排比冒泡更快?

冒泡的空间复杂度是O(n^2),应为他有两层for循环,
快排,选择基点,两边分别递归排序,空间复杂度是O(nlogn)明显nlogn < n^2
